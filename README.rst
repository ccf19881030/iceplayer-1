介绍
=====

  因为原有的iceplayer是一个好项目，但现在为其增加功能时遇到了
麻烦，因为原有代码耦合过度，难以从中分离部分有价值的代码，所以
在这里的项目就是为解决这个问题而诞生的。

  在GitHub的这个项目仅是用于为iceplayer重构代码的，我们可能不会
使用过多的原有代码。但会尽力提高iceplayer的可用性和效率。
